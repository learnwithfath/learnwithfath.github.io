Start Server:
  ./serve.sh

Create New Codelab:
  ./new-codelab.sh

Export All Codelabs:
  npm run export

Export Single Codelab:
  cd codelabs
  go run github.com/googlecodelabs/tools/claat@latest export filename.md

Stop Server:
  kill $(lsof -ti:9090)
